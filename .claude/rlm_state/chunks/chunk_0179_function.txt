## Error Handling

### Specific Exceptions

**Good:**
```python
try:
    result = parse_json(data)
except json.JSONDecodeError as e:
    logger.error(f"Failed to parse JSON: {e}")
    raise ValueError("Invalid JSON format") from e
```

**Bad:**
```python
try:
    result = parse_json(data)
except Exception:  # Too broad
    pass  # Silently fails
```
