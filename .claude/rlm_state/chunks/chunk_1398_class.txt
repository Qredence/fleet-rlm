## PUT vs PATCH

### PUT (Complete Replacement)
```json
PUT /users/1
{
  "name": "John",
  "email": "john@example.com",  // Must include ALL fields
  "age": 30
}
```

### PATCH (Partial Update)
```json
PATCH /users/1
{
  "name": "John"    // Only changed fields
}
```
